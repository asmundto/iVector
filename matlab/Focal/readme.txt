FoCal Multi-class Toolkit, version 1.0

Contents:
1. A manual:
../documentation/FoCal_MultiClass_Manual.pdf

2. Several sub-directories of MATLAB m-files.
../matlab/multifocal/.. 
To install, copy the sub-directories to a suitable location 
and set your MATLAB path to include all of these subdirectories.

Compatibility:
These m-files have been tested to run on versions 5 and 7 of MATLAB.

Demo:
After installation, Type at the MATLAB prompt:
>> multifocal_example1_fusion
